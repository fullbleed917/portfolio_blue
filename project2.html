Web Design - Branding

STREET DREAMZ

Role: Webdesigner

Agency: Freelance Project

Date: 2018

INTRODUCTION

Street Dreamz is a Maryland,DC based comprehensive automotive customization shop specializing in aftermarket wheels and audio accessories

Previous Site Analysis

Client used an economy template website, creating an undifferentiated, uncompetitive and undesirable user experience. The online prescence was erratic, lacked a cohesive brand identity, resulting in poor connection with customers and an unprofessional experience.

Wireframes and Prototypes

Given aspects such as the content and buisness goals, several usability reviews were performed with the goal in mind as to where the user would end up. We could then structure the site towards this interaction flow.

Moodboard collection

Created to establish a visual key for the client to understand the graphic elements, color, typography, look and feel. To set a realistic expectation and communicate effectivly the design of the final product.
